виды соединения таблиц
1. INNER JOIN ->  общее пересечение (пересечение 2-х кругов) - посередине
2. LEFT JOIN -> левый круг полностью, остаток части правого круга
3. RIGHT JOIN -> правый круг полностью, остаток части левого круга
4. FULL JOIN -> полностью оба круга

Задачи
1. получение данных из нескольких таблиц
2. соединение таблиц по общим столбцам
3. получение данных из связанных таблиц
4. фильтрация данных по условию
